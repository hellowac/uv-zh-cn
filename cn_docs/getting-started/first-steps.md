# uv 初始步骤

在完成 [uv 安装](./installation.md) 后，可以通过运行 `uv` 命令检查 uv 是否可用：

```console
$ uv
An extremely fast Python package manager.

Usage: uv [OPTIONS] <COMMAND>

...
```

您应该会看到一个列出可用命令的帮助菜单。
